# max_openears

## Short tutorial

### Hardware
Make sure you have the following:
- Raspberry Pi 4 or newer
- Microphone. We use a UMIK.
- Antenna to receive WiFi
- Casing
- (old) CORAL USB TPU (without it is not fast enough)

### Software
The file is in the following path:
``/home/pi/ ``

Run it with:
``python .py``
